#Comparing-Analysis-Method-Implementations-in-SAS-and-R

This repository contains R and SAS codes used for the statistical analyses in my thesis.

## Repository Contents

## 1. Accelerated Failure Time (AFT) Models

Contains:
- Loglogistic AFT model
- Lognormal AFT model
- Weibull AFT model
- Exponential AFT model
- Generalized Gamma model
- Generalized F model
- Cox-snell residual plots
- Standardized residual plots

Folder:
AFT R-SAS-codes


## 2. Non-PH Tests R-SAS codes

Contains:
- Weighted log-rank tests
   - Peto-Peto
   - Modified Peto-Peto
   - Tarone-ware
   - Gehan-Breslow/Wilcoxon
   - Fleming-Harrington
- Maxcombo test
   - Unstratified MaxCombo
   - Stratified MaxCombo
- Modestly weighted log-rank test
   - Unstratified MWLRT
   - Stratified MWLRT
  
Folder:
Non-PH R-SAS cpdes
